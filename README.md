# 畫派辨識與風格轉換系統
本研究主要專注於七個畫派的處理，分別是巴洛克、立體派、盛世文藝復興、印象派、極簡主義、普普藝術及浮世繪。最終畫派辨識模型的整體辨識準確率約有85%，而風格轉換的部分，平均 Gram Matrix 差異為 1.55，SSIM 約為 0.30，整體呈現良好的風格轉換與內容結構保留。系統亦具備操作介面與畫派知識介紹，適用於藝術教育與創作輔助。
